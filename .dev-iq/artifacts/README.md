# Dev.IQ Artifact Store

Local session artifacts generated by Dev.IQ skills. All content here is
gitignored — nothing in this directory is ever committed.

## Structure

| Directory | Skills that write here |
|-----------|----------------------|
| `adrs/` | `/generate-adr` |
| `rollback-plans/` | `/generate-rollback-plan` |
| `onboarding-guides/` | `/onboard-codebase` |
| `openapi-specs/` | `/generate-openapi` |
| `pr-reviews/` | `/review-pr-readiness`, `/code-review` |
| `signals/` | DI signal records (Phase 2) |

## Usage

**Claude Code:** Skills can write directly using the Write tool. When a skill
generates an artifact, confirm with "save it" and it will be written here.

**Copilot Chat:** Skills write via the filesystem MCP server when connected.
If MCP is not wired, copy the output from chat and save manually.

Artifacts are yours to review, edit, and apply. The `@di-review-required`
marker on skill output means the content has not yet been human-reviewed.
